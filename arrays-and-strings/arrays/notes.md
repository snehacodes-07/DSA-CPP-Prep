ARRAYS :

data structure --> fixed size collection of same type elements
stored in contiguous memory
access O(1)
size fixed
length -->  sizeof(array) ---> whole size of array
               sizeof(datatype) ---> size of element

   sizeof(array)/sizeof(datatype) ----> NO. OF ELEMENTS
 

VECTOR ARRAYS :


variable size
operations:   
  
push_back()  - add element at end
pop_back()   - remove last element
insert()     - insert element at specific position
erase()      - remove from specific position
clear()      - remove all element , create empty vector
assign()     - assign new content , replace current content
swap()       - exchange contents of two vectors
size()       - no. of elements
empty()      - true if empty, false otherwise
