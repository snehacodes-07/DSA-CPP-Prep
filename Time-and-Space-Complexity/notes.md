data structure :  used to store and organise data in a way that it allows efficient access and manipulation 

algorithm :  procedures and technique used to manipulate and process data

time complexity : describes how execution time grows as the input size increases. quantifies number of operations

asymptotic notations :   best case (omega)
                         avg case (theta)
                         worst case (O)

**rules:
         O(constant) ---> O(1)
         O(n+5) ----> O(n)
         O(n^2+n+5) ----> O(highest order)

different for loop --> add
nested for loop --> multiply


types of time complexity :

linear : O(n)
constant : O(1)
quadratic : O(n^2)
cubic : O(n^3)
logarithmic : O(log n)
exponential : O(2^n) 


comparison : O(1) < O(log n) < O(n) < O(n log n) < O(n^2) < O(2^n) < O(n!)
             const    log     linear               quad      expo    fact


space complexity : how much space is used based on inout value
